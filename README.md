# 📈 EquiTrade Stock App

> A modern stock trading and analysis application for tracking markets, analyzing trends, and making smarter investment decisions.

---

## 🚀 Features

- 📊 Real-time stock data tracking  
- 📈 Interactive charts and visualizations  
- 🔍 Stock search and filtering  
- 💼 Portfolio tracking  
- 🧠 Technical analysis tools  
- ⚡ Fast and responsive UI  

---

```bash
Frontend: React / HTML / CSS
```

## ⚙️ Installation

### 1. Clone the repository
```bash
git clone https://github.com/killengary37/equitrade_stock_app.git
cd equitrade_stock_app
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the app
```bash
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file in the root directory and add:

```env
API_KEY=your_api_key_here
```
